# ChangeLog

## 1.0.3 @ 2019-2-15

* 修改 sf/index.html 中的 JS 写法，排除 ES6，避免对旧版本手机造成影响

## 1.0.2 @ 2018-9-26

* 修改一处样式问题，避免在刷新时输入部分和 SF 部分同时出现。（需要强制清除缓存）

## 1.0.1 @ 2018-9-7

* 初始版本
  * 提供命令 `mip2 sf` 启动 SF 线下调试服务器
  * 支持参数 `-p` 或者 `--port` 修改端口号
  * 支持参数 `-o` 或者 `--autoopen` 自动打开页面
  * 通过 `http://localhost:8210/sf` 访问页面
